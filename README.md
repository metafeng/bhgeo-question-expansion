# bhgeo-question-expansion

**GEO 用户问题拓展工具** — 输入公司/产品资料，自动生成从关键词到内容布局的完整 GEO 策略。

---

## 这个 Skill 解决什么问题

GEO 内容布局的核心逻辑是：先知道用户会问什么，再决定做什么内容。

传统 SEO 靠关键词工具挖掘搜索词，GEO 的战场是 ChatGPT、Perplexity、豆包、Kimi 这类 AI 搜索引擎。用户在这里不是输入关键词，而是用自然语言提问。内容团队需要预测的不是「词」，而是「问题」。

这个 Skill 一次输入，完成两件事：
1. **GEO 收录监测**：生成 L1→L2→L3 结构化关键词，用于检测 AI 是否在回答相关问题时提及了品牌
2. **内容策略规划**：生成 L4 真实用户问法，评分筛选，给出内容形式和平台布局建议

---

## 7 模块工作流

```
输入：公司/产品资料
  ↓
模块1：L1 核心词识别        ← 从资料中提炼 3-5 个核心品类词
  ↓
模块2：L2 标准拓展词生成    ← 形容词 + L1，每个 L1 生成 8-10 个
  ↓
模块3：L3 核心搜索问题生成  ← 前缀 + L2 + 后缀，用于 GEO 收录监测
  ↓
模块4：L4 真实用户问法改写  ← L3 → 自然语言，用于内容策略
  ↓
模块5：购买意图与转化评分   ← 意图/阶段/P0-P2/内容形式
  ↓
模块6：Top 20 优先布局问题  ← 按优先级筛选最值得布局的 20 个
  ↓
模块7：内容形式与平台建议   ← 针对 P0 问题给出内容框架和平台推荐
```

---

## 层级定义

| 层级 | 定义 | 用途 | 示例 |
|------|------|------|------|
| **L1** | 核心品类词/产品词 | 拓展起点 | AI数字人、GEO服务 |
| **L2** | 形容词 + L1 | GEO 监测词 | 专业AI数字人、性价比GEO服务 |
| **L3** | 前缀 + L2 + 后缀 | GEO 收录监测 | 2026年专业AI数字人排行榜 |
| **L4** | L3 的自然语言改写 | 内容策略 | 国内AI数字人哪家最好用？ |

---

## 输出结构

一次运行输出 7 个部分：

```
一、L1 关键词（3-5 个）
二、L2 拓展词 + L3 核心问题（按 L1 分组，含品牌词标记）
三、L4 真实用户问法（每 L1 生成 6-10 个）
四、购买意图与转化匹配度评分（全量问题评分表）
五、Top 20 优先布局问题（排名列表）
六、内容形式与平台布局建议（针对 P0 问题）
七、统计汇总
```

## 如何使用

### 触发词

直接在对话中说以下任意一个词即可触发：

> GEO 问题拓展、用户问题分析、内容布局规划、问题矩阵、GEO 内容策略、帮我拓展问题、挖掘用户问题、AI 搜索问题、生成拓展词、关键词拓展、GEO 关键词

### 输入方式一：自由文本（推荐）

直接粘贴公司介绍、产品说明、官网文案：

```
必火AI 是一家专注 AI 数字人技术的公司，核心产品是 AI 短视频数字人创作平台，
支持一站式短视频创作：定位、选题、文案、数字人拍摄、智能剪辑、发布。
核心优势：唇形匹配效果好、声音克隆相似度高、使用方便。
目标客户：企业品牌方、内容创作者、个人IP、电商卖家。
品牌词：必火AI
```

### 输入方式二：结构化填写

```
行业：AI 技术 / 短视频内容创作
核心产品：AI 短视频数字人创作平台（定位→选题→文案→拍摄→剪辑→发布一站式）
目标客户：企业品牌方、内容创作者、个人IP、电商卖家
竞争优势：唇形匹配效果好、声音克隆相似度高、使用方便
用户痛点：真人出镜成本高、拍摄周期长、内容更新慢
品牌词：必火AI
```

## 优先级评分说明

| 优先级 | 定义 | 典型问题类型 |
|--------|------|------------|
| **P0** | 搜索量大 + 决策影响强 + 与品牌优势高度匹配 | 比较型、价格型、推荐型 |
| **P1** | 搜索量中等 或 决策影响中等 | 信息型、认知型 |
| **P2** | 长尾精准问题，搜索量小但覆盖特定场景 | 操作型、故障型 |
